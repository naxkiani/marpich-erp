# Municipality — local government bounded context

## Ubiquitous language

| Term | Definition |
|------|------------|
| **Permit** | Local license (construction, business, etc.) |
| **Service request** | Citizen case tracked by municipality |
| **Utility account** | Water/waste/electricity billing account |

## Independence

- **Not** merged with `government` (national/federal scope)
- Own schema `municipality_*`

## API

Prefix: `/api/v1/municipality`
